2.3.3. Drawing¶

Code for drawing various things.

teaspoon.TDA.Draw.drawDgm(D, boundary=None, epsilon=0.5, color=None)[source]¶

Draws simple persistence diagram plot

Parameter D

a persistence diagram, given as a Kx2 numpy array

Parameter boundary

Parameter epsilon

The diagram is drawn on [0,boundary]x[0,boundary]. If boundary not given, then it is determined to be the max death time from the input diagram plus epsilon.

teaspoon.TDA.Draw.drawPtCloud(P)[source]¶

Draws simple point cloud plot

Parameter P

a point cloud, given as a KxD numpy array. Even if D>2, only the first two coordinates are plotted.
